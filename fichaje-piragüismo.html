📋 Cómo conectar con Google Sheets

  1. Abre tu Google Sheet de fichajes
  2. Ve a Extensiones → Apps Script
  3. Borra el código y pega el código de Apps Script
  4. Pulsa Implementar → Nueva implementación
  5. Tipo: Aplicación web · Acceso: Cualquier persona
  6. Copia la URL de implementación y pégala arriba

📱 Instalar en Android como app

  1. Abre esta página en Chrome
  2. Pulsa los 3 puntos del menú
  3. Selecciona "Añadir a pantalla de inicio"
  4. Confirma · Ya tienes el icono en el escritorio
